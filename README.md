# Termux-root
![Termux-root preview](Termux-root.jpg)

## introduction
Termux-root is a tool that has the ability to provide root access to Termux users even though their device has not been rooted

## Instalations
```
$ pkg update -y && pkg upgrade -y
$ pkg install git
$ pkg install python -y
$ git clone https://github.com/Whomrx666/Termux-root.git
$ cd Termux-root 
$ python Termux-root.py
```

## To check root access
```
$ whoami
```

## Instructions
- **first**: Install tools according to the instructions above
- **second**: Select no 1 to start rooting Termux
- **third**: The tool will start the rooting process on Termux
- **last**: Press enter to start running root access in termux
